# Skrumble

## Beep Server
Backend for new Skrumble Communication API.

## Setup Instructions
Checkout the repository and run script `deploy.sh`
```
git clone https://github.com/https://github.com/skrumblenetwork/beep-server.git
chmod +x deploy.sh
./deploy.sh
```
